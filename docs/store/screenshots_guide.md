# スクリーンショット準備ガイド

## 必要なスクリーンショット

Google Playには最低2枚、最大8枚のスクリーンショットが必要です。

### 推奨サイズ
- 縦向き: 1080 x 1920 px（16:9）
- 横向き: 1920 x 1080 px

### 撮影すべき画面

| # | 画面 | 説明 | ポイント |
|---|------|------|----------|
| 1 | マップ画面（記録中） | メイン機能 | 速度、移動手段、ドットが見える状態 |
| 2 | マップ画面（ドット表示） | ゲーム要素 | 複数のドットが表示されている |
| 3 | スコア画面 | スコアリング | 合計スコア、ドット収集数が見える |
| 4 | 履歴画面 | 履歴機能 | 複数の記録が表示されている |
| 5 | メダル画面 | 実績システム | 獲得済みメダルがある状態 |
| 6 | ダークモード | UI対応 | ダークモードの見た目 |

### 撮影手順

1. 実機またはエミュレータでアプリを起動
2. デモ用のデータを作成（記録を数回実行）
3. 各画面をスクリーンショット
4. 必要に応じてステータスバーをクリーンに

### ファイル保存先
```
docs/store/screenshots/
├── 01_map_tracking.png
├── 02_map_dots.png
├── 03_score.png
├── 04_history.png
├── 05_medals.png
└── 06_dark_mode.png
```

## 注意事項

- 個人情報（実際の住所など）が映らないように注意
- テスト用のダミーロケーションを使用推奨
- 見栄えの良いスコア・データで撮影
