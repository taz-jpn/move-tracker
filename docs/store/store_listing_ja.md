# Google Play ストア掲載情報（日本語）

## アプリ名
MoveTracker

## 短い説明（80文字以内）
移動を記録してスコア化！徒歩・自転車・車の移動をトラッキングし、ドットを集めてポイントを獲得

## 詳細な説明

🚶 移動するだけでポイントが貯まる！

MoveTrackerは、あなたの日常の移動をゲーム感覚で楽しくするアプリです。GPSで移動を自動記録し、移動手段に応じてスコアを獲得できます。

【主な機能】

📍 リアルタイム移動記録
・GPSで現在地を追跡
・移動距離、時間、速度をリアルタイム表示
・移動手段（徒歩/自転車/車）を自動判定

🎮 ドット収集ゲーム
・マップ上のドットを集めてボーナスポイント獲得
・ゴールド、シルバー、ノーマルの3種類のドット
・パックマン風の楽しいゲーム体験

🏆 スコアリングシステム
・徒歩: 3倍ポイント
・自転車: 2倍ポイント
・車・列車: 1倍ポイント
・エコな移動ほど高得点！

🎖️ メダル獲得
・累計距離、連続記録日数などで13種類のメダルを獲得
・マラソナー、ウィークリーチャンピオンなど様々な実績

📊 履歴管理
・過去の記録を一覧表示
・日別、週別の移動データを確認

【特徴】
✅ 完全オフライン対応 - データは端末内のみに保存
✅ プライバシー重視 - 外部サーバーへのデータ送信なし
✅ バッテリー効率 - 最適化されたGPS処理
✅ ダークモード対応

毎日の通勤、散歩、サイクリングを楽しくトラッキング！
今すぐダウンロードして、移動をゲームに変えましょう！

MoveTrackerは移動記録とゲーミフィケーションアプリです。GPSを使用してユーザーの移動（徒歩/自転車/車）を記録し、移動時間をスコアに変換します。

## 位置情報アクセス (500文字以内)

```
移動記録機能にバックグラウンド位置情報アクセスが必要です。

ユーザーは通勤、運動、旅行を記録するためにトラッキングセッションを開始します。アプリはGPS位置を継続的に監視し、移動距離、現在速度、移動手段（徒歩/自転車/車）を計算します。

バックグラウンドアクセスが必要な理由:
• セッションは長時間の通勤や運動中に数時間続くことがある
• 画面オフやアプリ最小化時も記録を継続する必要がある
• 正確な距離計算には途切れないGPSデータが必要

ユーザーはいつでも記録を停止できます。
```

## Video Instructions (YouTube URL required)

バックグラウンド位置情報権限の審査のため、機能デモ動画が必要です。

### 動画の内容

1. アプリを起動してトラッキングを開始
2. ホームボタンでアプリをバックグラウンドに
3. 画面オフの状態でしばらく移動（または待機）
4. アプリに戻って、バックグラウンドでも記録が継続していたことを確認

### 作成手順

1. 実機でスクリーン録画を開始
2. MoveTrackerを起動 → 記録開始
3. ホームに戻る（バックグラウンドへ）
4. 数秒〜数十秒待つ
5. アプリに戻る → 距離・時間が増えていることを確認
6. 録画を停止
7. YouTubeに「限定公開（Unlisted）」でアップロード
8. そのURLをPlay Consoleに入力

### 注意点

- 限定公開（Unlisted）でOK（公開不要）
- 30秒〜2分程度で十分
- 実際の位置情報が映る場合はプライバシーに注意
