# リリースノート v1.0.0

Play Console に入力するためのリリースノートです。

---

## 日本語 (ja-JP)

```
MoveTracker 初回リリース

移動を記録してスコア化する新しいアプリです。

主な機能:
• GPSで移動をリアルタイム記録
• 移動手段（徒歩/自転車/車）を自動判定
• 移動時間に応じてスコアを獲得（徒歩3倍、自転車2倍）
• マップ上のドットを集めてボーナスポイント
• 13種類のメダル実績システム
• 履歴管理で過去の記録を確認

特徴:
• 完全オフライン - データは端末のみに保存
• プライバシー重視 - 外部サーバーへの送信なし
• ダークモード対応
```

## 英語 (en-US)

```
MoveTracker Initial Release

A new app that tracks your movement and turns it into scores.

Key Features:
• Real-time GPS movement tracking
• Auto-detect transport mode (walking/cycling/vehicle)
• Earn scores based on travel time (3x walking, 2x cycling)
• Collect dots on the map for bonus points
• 13 medal achievements to unlock
• View your history and past records

Highlights:
• Fully offline - data stored only on device
• Privacy first - no external server communication
• Dark mode support
```
